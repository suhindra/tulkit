# Mini WebFormatter (prototype)

Prototype sederhana untuk membuat web formatter mirip https://webformatter.com.

Cara pakai:

- Buka file `index.html` di browser (double-click atau buka dari web server lokal).
- Paste kode di area kiri, pilih bahasa, lalu klik `Format`.
- Gunakan `Copy` atau `Download` untuk menyimpan hasil.

Libraries yang dipakai via CDN:

- `js-beautify` — format HTML / CSS / JS
- `sql-formatter` — format SQL
- `highlight.js` — syntax highlighting

Langkah berikut yang bisa saya bantu:

- Tambahkan server kecil (Express) untuk hosting dan API (optional).
- Tambahkan lebih banyak bahasa / opsi konfigurasi formatter.
- Buat layout dan UI lebih mirip webformatter.com (tabs, options panel).
